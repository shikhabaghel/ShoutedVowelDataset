This dataset contains shouted and normal speech along with EGG signal. Recordings are done in a controlled environment. 


1- Num of Speakers : 11 (6 Males and 5 Females)

2- Speakers were asked to utter vowels along with stop consonants at beginning and ending of each vowel.
	Ex : "kak", "kek", "kik", "kok" and "kuk"

3- Sampling Rate : 48 kHz
